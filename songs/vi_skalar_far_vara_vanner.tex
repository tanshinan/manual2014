\visa{Vi skålar för våra vänner}
{\footnotesize\textit{Melodi: Flickan går i ringen}}\par
\vspace{10pt}
Vi skålar för våra vänner,\\
och dom som vi känner,\\
och dom som vi inte känner,\\
dom skiter vi i!\\
\\
Vi skiter i våra vänner,\\
och dom som vi känner,\\
och dom som vi inte känner,\\
dom skålar vi för!
